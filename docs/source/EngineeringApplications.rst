工程应用 Engineering Applications
=================================

WOEAI 将建筑结构抗风、海上漂浮风电和 AI 赋能工程建模研究用于公开可说明的工程问题。本页按研究方向的两级分类组织应用场景，并只展示公开项目类别、问题类型和研究证据；未公开确认的合作方名称、当前合作状态和项目细节不在此展开。

应用场景 Application Scenarios
------------------------------

建筑结构抗风
~~~~~~~~~~~~

数值风洞与湍动入流
^^^^^^^^^^^^^^^^^^

- 城市风环境、复杂地形风场和工程场址风环境评估；
- 大气边界层湍动入流、LES 数值风洞和风场一致性生成；
- 面向规划片区、工程场址和建筑群风环境问题的 CFD 与 AI 赋能预测。

公开证据：:ref:`building-wind-project-support`、:ref:`research-direction-structural-wind`、:doc:`PublicationsByResearch`。

高层建筑抗风与优化
^^^^^^^^^^^^^^^^^^

- 高层建筑风荷载、风致响应和气动外形优化；
- 风洞试验、风振分析、减振控制和强风作用下结构响应评估；
- 输电塔线体系等结构强风作用问题的历史项目证据。

公开证据：:ref:`building-wind-project-support`、:ref:`engineering-enterprise-project-evidence`、:ref:`research-direction-structural-wind`、:doc:`Publications`。

海上漂浮风电
~~~~~~~~~~~~

浮式风机系统一体化分析与优化
^^^^^^^^^^^^^^^^^^^^^^^^^^^^

- 面向多设计阶段的浮式风机系统分析、优化和长期动力响应评估；
- 风机、平台、系泊和环境荷载耦合下的设计参数敏感性分析；
- 浮式风机运动与振动控制相关工程问题。

公开证据：:ref:`floating-wind-project-support`、:ref:`engineering-enterprise-project-evidence`、:ref:`research-direction-floating-wind`、:doc:`PublicationsByResearch`。

浮式混凝土平台结构设计
^^^^^^^^^^^^^^^^^^^^^^

- 钢筋混凝土、FRP-钢复合筋和海水海砂混凝土半潜式平台结构设计；
- 浮式基础初步方案、结构优化和材料体系相关工程问题；
- 混凝土支撑结构与钢结构方案的动力响应比较。

公开证据：:ref:`floating-wind-project-support`、:ref:`engineering-enterprise-project-evidence`、:ref:`research-direction-floating-wind`。

数值风浪流水池
^^^^^^^^^^^^^^

- 风、浪、流联合作用下浮式风机动力响应分析；
- 风洞和水槽模型试验、等效静力波浪荷载评估；
- 面向半潜式浮式风机的数值风浪流水池和模型验证。

公开证据：:ref:`floating-wind-project-support`、:ref:`engineering-enterprise-project-evidence`、:ref:`research-direction-floating-wind`。
