## 关键事实证据定位记录

- 摘要:
  - 文章使用: 中文摘要忠实翻译英文摘要，不附英文原摘要。
  - 证据位置: Zotero `abstractNote`; PDF file page 1 abstract.
- 核心结论: 本文提出 built-in rectangular pole TLD，用矩形立柱增强液体耗能，并为大型液舱提供内部支撑构件。
  - 证据位置: PDF file page 1 abstract; PDF file pages 2-3, Section 1.
- 核心结论: 传统纯水 TLD 固有阻尼偏低，论文引用典型量级约为 `0.5%`，因此需要内部阻挡装置增强耗能。
  - 证据位置: PDF file page 2, Section 1.
- 核心结论: OpenFOAM 两相流求解器通过耦合 level set 与 VOF 方法形成 CLS-VOF 路线，以提高自由液面捕捉精度并抑制伪流。
  - 证据位置: PDF file pages 3-4, Section 2.1-2.2, Eqs. (1)-(9).
- 核心结论: CLS-VOF 验证包括破坝液体、带孔板液舱和挡板液舱实验对比；在波高和壁面压力预测中，CLS-VOF 相比 VOF 降低平均相对误差。
  - 证据位置: PDF file pages 4-7, Section 2.3, Figs. 1-6.
- 核心结论: 参数数据库覆盖液体充填高度、立柱数量、立柱安装位置、立柱阻塞率和激励幅值；立柱阻塞率使用 $\Theta=a/B$。
  - 证据位置: PDF file pages 5-8, Section 3, Table 1, Figs. 7-8.
- 核心结论: 随着立柱阻塞率 $\Theta$ 增大，附加质量效应更明显，TLD 内部液体固有频率降低，阻尼比提高。
  - 证据位置: PDF file pages 5-9, Section 3.1, Figs. 8-10.
- 核心结论: 立柱位置对频率影响较小，但对阻尼影响明显；立柱从舱壁向中部移动时，尾流耦合和剪切层增强，阻尼提高。
  - 证据位置: PDF file pages 8-10, Section 3.2, Figs. 11-12.
- 核心结论: 在总阻塞率保持不变时，把较大立柱细分为多根小立柱会增加流动通道、减弱涡脱落强度，并可能降低阻尼系数。
  - 证据位置: PDF file pages 9-11, Section 3.3, Figs. 13-14.
- 核心结论: 激励幅值 $A$ 增大时，自由液面出现卷起、破碎和更强非线性，涡脱落和湍动区域增强，TLD 阻尼比提高。
  - 证据位置: PDF file pages 9-12, Section 3.4, Figs. 15-16.
- 核心结论: PSO 拟合的等效机械模型用于估计带内置矩形立柱 TLD 的晃荡频率和阻尼比；晃荡频率模型拟合结果为 $Q_1=-0.265$、$Q_2=2.645$、$R^2=0.984$、$r=0.998$。
  - 证据位置: PDF file pages 10-13, Section 4.1, Eqs. (13)-(17), Fig. 17.
- 核心结论: 阻尼比等效模型拟合结果为 $K_1=0.517$、$K_2=0.414$、$K_3=0.283$、$K_4=0.123$、$K_5=-0.351$、$K_6=0.493$、$R^2=0.872$、$r=0.934$。
  - 证据位置: PDF file page 13, Section 4.2, Eq. (18), Fig. 18.
- 核心结论: 与基于势流理论的理论解析模型相比，本文等效机械模型与 CLS-VOF 数值模拟更接近；理论模型因线性化和小幅激励损失系数而容易高估阻尼。
  - 证据位置: PDF file pages 13-15, Section 4.3, Figs. 19-21.
- 核心边界: 本文主要研究矩形液舱、内置矩形立柱和单向简谐激励；后续将扩展到不同立柱形状、随机和多向激励，以及完整海洋结构模型。
  - 证据位置: PDF file pages 14-15, Section 5 final paragraph.
