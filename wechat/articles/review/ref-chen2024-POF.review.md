## 关键事实证据定位记录

- 摘要:
  - 文章使用: 中文摘要忠实翻译英文摘要，不附英文原摘要。
  - 证据位置: Zotero `abstractNote`; PDF file page 1 abstract.
- 核心结论: 本文比较 IMC、DFC 和 LPC 三类 RAPF 方法在高层建筑风荷载 LES 中的表现。
  - 证据位置: PDF file page 1 abstract; PDF file pages 3-5, Section II.
- 核心结论: 未修正入口质量失衡会在空域中造成显著人工压力脉动，建筑位置附近可达到动压约 `50%`。
  - 证据位置: PDF file pages 7-8, Section III.C, Figs. 6-7.
- 核心结论: IMC 与 DFC 能在整个流场内有效减小不真实压力脉动；LPC 能降低压力参考点附近的局部非物理压力脉动，但距离参考点越远效果越弱。
  - 证据位置: PDF file pages 7-8, Section III.C, Figs. 6-7.
- 核心结论: IMC 与 DFC 会改变初始湍流场，使其满足入口质量平衡或无散条件；LPC 不修改原始入口湍流，因此更能保持初始湍流特性。
  - 证据位置: PDF file pages 8-11, Section III.D, Figs. 8-13.
- 核心结论: 未修正建筑算例在 `2/3H` 高度处的脉动风压系数约为动压的 `50%-70%`；采用 RAPF 方法后约为 `10%-40%`。
  - 证据位置: PDF file page 13, Section IV.C, Fig. 15.
- 核心结论: 建筑表面压力系数标准差的平均误差中，未修正算例为 `108%`，RAPF 算例约为 `10.8%-16.3%`；峰值压力中未修正算例所有表面平均误差为 `12.0%`，HB2-HB5 约为 `5%`，HB6 为 `6.7%`。
  - 证据位置: PDF file pages 16-18, Section IV.C, Figs. 20-23.
- 核心结论: 对称建筑的整体基底力和力矩可能抵消部分非物理压力脉动，因此整体力/力矩看似合理不代表局部表面风压可信。
  - 证据位置: PDF file pages 18-20, Section IV.D, Tables IV-V, Fig. 24; PDF file page 21 conclusion.
- 核心边界: 结论依赖 CDRFG 合成湍流入口、TPU 方形截面高层建筑数据库、OpenFOAM v2206 和本文采用的网格/边界条件设置；不应无条件推广到所有入口生成方法或建筑外形。
  - 证据位置: PDF file pages 6-7, Section III.B; PDF file pages 12-13, Section IV.A.
- 关键图:
  - Fig. 1 `Schematic diagram of the application of inflow turbulence`: PDF file page 4; used as article Figure 1; asset `wechat/assets/public-safe/ref-chen2024-POF/fig01.png`.
  - Fig. 7 `Comparisons of the mean and STD profiles of pressure coefficients along X direction`: PDF file page 8; used as article Figure 7; asset `wechat/assets/public-safe/ref-chen2024-POF/fig07.png`.
  - Fig. 11 `NMAE of turbulence properties along X direction`: PDF file page 10; used as article Figure 11; asset `wechat/assets/public-safe/ref-chen2024-POF/fig11.png`.
