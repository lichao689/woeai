技术合作 Technical Collaboration
========================================

WOEAI 面向风工程、海上风电和 AI 赋能工程建模中的复杂问题开展技术合作与工程实践。合作页面以公开项目类别和能力方向呈现，不发布未经确认的合作方名称或未公开项目细节。

技术合作方向 Capability Areas
--------------------------------

建筑结构抗风
~~~~~~~~~~~~

- 数值风洞与湍流入流，包括大气边界层湍流生成、LES 入流和城市/复杂地形风场；
- 高层建筑抗风与优化，包括风致荷载、结构响应、气动外形优化和减振控制；
- 面向建筑风场、建筑几何和数值模拟流程的 AI 赋能建模方法。

海上漂浮风电
~~~~~~~~~~~~

- 浮式风机系统一体化分析与优化；
- 浮式混凝土平台结构设计；
- 数值风浪流水池、风浪流联合作用和动力响应分析；
- 浮式风机运动和振动控制。

项目证据 Project Proof
----------------------

如果需要先按问题类型判断合作相关性，可从 :doc:`Research`、:doc:`BuildingStructuralWindResistance` 和 :doc:`FloatingOffshoreWindEnergy` 查看相关研究脉络；如果需要完整核验，公开项目证据库见 :doc:`Projects`。

项目记录包括政府项目和企业委托项目，覆盖浮式风机系统优化、高层建筑气动优化、输电线路微尺度台风风场、风洞试验及风振分析等方向。
