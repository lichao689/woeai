技术合作 Technical Collaboration
========================================

WOEAI 面向风工程、海上风电和 AI 赋能工程建模中的复杂问题开展技术合作与工程实践。合作页面以公开项目类别和能力方向呈现，不发布未经确认的合作方名称或未公开项目细节。

技术合作方向 Capability Areas
--------------------------------

结构抗风与风环境
~~~~~~~~~~~~~~~~

- 高层建筑气动外形优化与风致荷载响应分析；
- 输电线路和输电塔在台风、复杂地形等条件下的风场与结构响应；
- 城市风环境与多尺度风场模拟；
- 大气边界层湍流生成和数值风洞方法。

海上风电与海洋工程结构
~~~~~~~~~~~~~~~~~~~~~~

- 浮式海上风机系统一体化分析与优化；
- 半潜式风机基础结构设计；
- 风浪联合模型试验与动力响应分析；
- 浮式风机运动和振动控制。

AI 赋能工程方法
~~~~~~~~~~~~~~~

- 面向结构响应预测的机器学习和图神经网络方法；
- 面向城市风场、建筑几何和数值模拟流程的智能建模方法；
- 面向工程结构优化的代理模型与数据驱动方法。

项目证据 Project Proof
----------------------

如果需要先按问题类型判断合作相关性，可从 :doc:`Research` 和三个具体方向页查看相关项目摘要；如果需要完整核验，公开项目证据库见 :doc:`Projects`。

项目记录包括政府项目和企业委托项目，覆盖浮式风机系统优化、高层建筑气动优化、输电线路微尺度台风风场、风洞试验及风振分析等方向。
