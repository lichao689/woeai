## 关键事实证据定位记录

- 摘要:
  - 文章使用: 中文摘要为英文原摘要的忠实翻译，不附英文原摘要。
  - 证据位置: Zotero Desktop Local API `abstractNote`; PDF abstract on `PDF file page 2`.
- 核心结论: 研究比较安装在混凝土 LCP3 和钢 HSP3 Y 型半潜平台上的两种浮式风机，采用 `1:60` 风浪联合模型试验和耦合多体仿真。
  - 证据位置: PDF abstract on `PDF file page 2`; Introduction and objective on `PDF file page 3`.
- 核心结论: HSP3 需要更多压载水达到相同吃水，重心低于 LCP3；重心差异是两种结构动力响应差异的主要原因。
  - 证据位置: system description and Table 1 on `PDF file page 4`; Conclusions on `PDF file page 20`.
- 核心结论: HSP3 横摇/纵摇固有周期比 LCP3 约短 `1.6 s`，在额定运行风况下平均纵摇约小 `0.5 deg`。
  - 证据位置: free-decay discussion and Table 6 on `PDF file page 10`; Conclusions on `PDF file page 20`.
- 核心结论: HSP3 在平台纵摇减小方面有优势，但波频响应中可能出现相反趋势，使塔底载荷和机舱加速度差异并不总是显著。
  - 证据位置: RAO discussion around Figure 12 on `PDF file page 11`; Conclusions on `PDF file page 20`.
- 核心结论: 在更接近真实运行的随机风浪条件下，两种平台塔底弯矩和机舱加速度差异整体较小，但不同统计量和工况仍显示不同方向的差别。
  - 证据位置: Figure 26 and surrounding discussion on `PDF file page 20`.
- 关键图:
  - Fig. 1 `Configuration of the semi-submersible floating wind turbine`: `PDF file page 4`; used as article Figure 1.
  - Fig. 10 `The framework for the FAST simulator`: `PDF file page 10`; used as article Figure 10.
  - Fig. 12 `Comparisons of RAOs between the LCP3 and HSP3 model`: `PDF file page 11`; used as article Figure 12.
