隐私说明 Privacy Notice
===========================

本页说明 WOEAI 网站对访问者和联系人的个人信息处理边界。本站为静态学术网站，不提供账户注册、在线表单、评论区、在线支付或文件上传功能。

本站不主动收集的信息
--------------------

本站不主动设置第三方跟踪脚本，不使用 Google Analytics 等访问统计工具，也不在本站自行建立访问者画像。

托管平台日志
------------

本站源代码托管于 GitHub，并通过 Read the Docs 等静态文档托管或发布服务提供页面访问。访问相关页面时，GitHub 或 Read the Docs 可能基于其平台安全、服务运行和日志管理需要处理访问日志、浏览器信息、访问时间、来源页面、IP 地址等基础技术信息。

这些平台日志不由本站直接控制，具体处理规则以相关平台的隐私政策和服务条款为准：

- `GitHub Pages data collection <https://docs.github.com/en/pages/getting-started-with-github-pages/about-github-pages#data-collection>`_
- `Read the Docs Privacy Policy <https://about.readthedocs.com/privacy-policy>`_

邮件和微信联系
--------------

如果你通过邮箱或微信二维码主动联系 WOEAI，来信、微信资料、联系方式、申请背景、研究兴趣、合作问题和随信材料等信息将仅用于以下目的：

- 回复你的咨询；
- 进行招生、博士后、科研训练或学术交流沟通；
- 判断工程应用、技术合作或工程咨询问题是否适合进一步讨论；
- 在双方后续沟通所必需的范围内保存沟通记录。

除非取得你的明确同意，或法律法规另有要求，本站不会将你通过邮件或微信主动提供的信息公开发布，也不会出售、出租或转交给无关第三方。

第三方视频嵌入
--------------

部分页面可能嵌入 Bilibili、YouTube 或其他第三方视频平台播放器。访问含有嵌入播放器的页面时，浏览器可能会向第三方平台请求播放器资源；播放、暂停、跳转或打开外部链接时，相关访问数据由第三方平台按其隐私政策和服务条款处理。

如不希望加载第三方播放器，可避免点击播放，并通过页面提供的外部链接自行决定是否访问第三方平台。

信息最小化提示
--------------

首次联系时，请尽量只提供与咨询目的直接相关的信息。请不要通过普通邮件或微信发送身份证件号码、家庭住址、银行账户、未公开项目数据、涉密材料、商业秘密或其他不必要的敏感信息。
